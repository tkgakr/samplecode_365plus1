# 0430: 6時間先気温予測サンプル

このディレクトリには、Julia で作成した時系列予測のサンプルスクリプト `code.jl` が含まれています。

このスクリプトは、人工的に生成した気温データを使って、過去の気温や時刻情報から **6時間先の気温** を予測します。学習には `MLJ` 経由で `XGBoostRegressor` を使用します。

## 前提条件

- Julia がインストールされていること
- インターネット接続があること
  - 初回実行時に不足しているパッケージを自動導入するため

## 実行方法

`code.jl` は、自分自身のあるディレクトリを Julia 環境として有効化し、不足パッケージがあれば自動でインストールするようになっています。

ターミナルでこのディレクトリに移動して、次のコマンドを実行してください。

```bash
julia code.jl
```

または、リポジトリのルートなど別の場所にいる場合は、ファイルパスを指定して実行できます。

```bash
julia 0430/code.jl
```

## 初回実行について

初回実行時は、以下のパッケージの導入に時間がかかることがあります。

- `MLJ`
- `DataFrames`
- `XGBoost`
- `MLJXGBoostInterface`

2回目以降は、すでに導入済みであればそのまま実行されます。

## 実行すると何が起こるか

スクリプトは次の流れで動作します。

- 1時間刻みの疑似気温データを生成する
- ラグ特徴量や移動平均を作る
- 6時間先の気温を目的変数として学習データを作る
- XGBoost 回帰モデルを学習する
- テストデータで予測し、RMSE を表示する

## 出力例

実行すると、最後に次のような形式で評価結果が表示されます。

```text
[ Info: Test RMSE (6h-ahead temp) = ...
```

`RMSE` は予測誤差を表す値で、一般に小さいほど予測精度が高いことを意味します。
